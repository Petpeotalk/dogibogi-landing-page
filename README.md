## 도기보기 랜딩 페이지

```bash
git clone petpeotalk.github.io
npm install
# node-sass 에러 날 경우
node node_modules/node-sass/scripts/install.js 또는 sudo npm install -g --unsafe-perm node-sass
npm run start
```

## 기술 스택

- gulp(자바스크립트 번들링 툴)
